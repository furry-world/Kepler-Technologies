K6 microprocessor

REGISTERS

------------

R. A, B, C, D, E, F and G - 6 bits - general purpose

R. H - 1 bit - used for the carry/underflow flag

Program Counter - 12 bits

Stack Pointer - 4 bits

Delay Timer - 6 bits

------------

OPCODES

------------

000000 CCCCCC CCCCCC - call subroutine at 12-bit program memory address and pusb current address to stack

000001 CCCCCC CCCCCC - jump to 12-bit program memory address

000010 CCCCCC - relative jump to current address + 6-bit offset

000011 - return from subroutine to the latest pushed address in stack and pop it off the stack

------

000100 XXXICC CCCCCC - skip the next instruction if the value of register XXX is equal to 8-bit constant (I determines immediate or address value)

000101 XXXICC CCCCCC - skip the next instruction if the value of register XXX is not equal to 8-bit constant (I determines immediate or address value)

000110 XXXICC CCCCCC - sets the value of register XXX to 8 bit constant (I determines immediate or address value)

000111 XXX0CC CCCCC - stores the value of register XXX to 8-bit address

001000 XXXYYY - skip the next instruction if value of register XXX is equal to value of register YYY

001001 XXXYYY - skip the next instruction  if value of register XXX is not equal to value of register YYY

001010 XXXYYY - sets the value of register XXX to value of register YYY

001011 XXXYYY - adds register YYY to register XXX (checks the flag for carry, and sets it to 1 if overflow occurs)

001100 XXXYYY - subtracts register YYY from register XXX (checks the flag for underflow and sets it to 0 if underflow occurs)

001101 XXXYYY - set register XXX to (XXX OR YYY)

001110 XXXYYY - set register XXX to (XXX AND YYY)

001111 XXXYYY - set register XXX to (XXX XOR YYY)

010000 XXX000 - shift register XXX to the left through carry

010001 XXX000 - shift register XXX to the right through carry
